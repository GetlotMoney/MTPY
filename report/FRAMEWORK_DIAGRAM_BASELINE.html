损失函数约束与意义

1. Seen 分类监督 \( \mathcal{L}_{ce} \)
\( \mathcal{L}_{ce}=\mathrm{CE}(z_{seen},y),\quad z_{seen}=logits_{200}[:,seen] \)
约束对象: 最终融合后的 seen 类分类分数。输入: \(z_{seen}\in\mathbb{R}^{B\times150}\), \(y\in\mathbb{R}^{B}\)。意义: 提供主分类监督, 让融合后的 \(base\_logits+0.3\cdot local\_score\) 能正确区分 150 个 seen 类。
2. 文本拓扑保持 \( \mathcal{L}_{topo} \)
\( \mathcal{L}_{topo}=1-\rho(\mathrm{vec}(C_{adapt}),\mathrm{vec}(C_{clip})),\quad C=TT^\top \)
约束对象: Adapter/条件化后的文本类别结构。输入: \(T_{adapt},T_{clip}\in\mathbb{R}^{200\times768}\), 相似度矩阵 \(C\in\mathbb{R}^{200\times200}\)。意义: 让文本可适配, 但不破坏 CLIP 原始类别间语义拓扑。
3. 全局-局部一致性 \( \mathcal{L}_{cons} \)
\( \mathcal{L}_{cons}=T^2D_{KL}(p_b^{det}\Vert p_l),\quad p=\mathrm{softmax}(z_{seen}/T) \)
约束对象: Cross-Modal 局部分支的 seen 类分布。输入: \(base\_logits[:,seen]\in\mathbb{R}^{B\times150}\), \(local\_score[:,seen]\in\mathbb{R}^{B\times150}\)。意义: 用全局 CLIP 分支当老师, 防止局部分支学出和全局语义相反的排序。
4. 双向分支互蒸馏 \( \mathcal{L}_{msdn} \)
\( \mathcal{L}_{msdn}=\frac{T^2}{2}[D_{KL}(p_s^{det}\Vert p_v)+D_{KL}(p_v^{det}\Vert p_s)] \)
约束对象: s2v 与 v2s 两条 Cross-Modal 分支。输入: \(score\_s2v[:,seen]\), \(score\_v2s[:,seen]\in\mathbb{R}^{B\times150}\)。意义: 两个方向互相当老师, 抑制单一路径塌缩, 让局部视觉到语义、语义到视觉两路判断保持一致。
5. AG-JEPA 正目标预测 \( \mathcal{L}_{jepa} \)
\( \mathcal{L}_{jepa}=1-\cos(\hat z,z_m^{det}),\quad \hat z=f_\theta(c,t_y) \)
约束对象: JEPA predictor 对 masked semantic patch 的预测。输入: context \(c\in\mathbb{R}^{B\times512}\), 类文本 \(t_y\in\mathbb{R}^{B\times512}\), masked target \(z_m\in\mathbb{R}^{B\times512}\)。意义: 让剩余 patch 和真实类文本能预测被遮住的关键语义 patch, 强化局部语义建模。
变量说明: \(f_\theta\) 是 JEPA predictor, 输入为拼接后的 \([c,t_y]\), 输出预测特征 \(\hat z\in\mathbb{R}^{B\times512}\)。\(c\) 是对视觉 patch 执行 mask 后保留下来的视觉上下文, 即去掉 top-8 target patch 后的剩余 patch_z 平均特征; \(t_y\) 是真实类别文本经过 embed_text 后的特征; \(z_m\) 是被 mask 的 top-8 semantic patch 平均特征, 在 loss 中 detach 作为预测目标。
6. AG-JEPA 负文本约束 \( \mathcal{L}_{neg} \)
\( \mathcal{L}_{neg}=\mathrm{ReLU}(\cos(\hat z_n,z_m)-s_+^{det}+m),\quad s_+=\cos(\hat z,z_m) \)
约束对象: 负类别文本对同一个 masked target 的解释能力。输入: 负文本 \(t_{neg}\in\mathbb{R}^{B\times512}\), target \(z_m\in\mathbb{R}^{B\times512}\), 正相似度 \(s_+\in\mathbb{R}^{B}\)。意义: 如果负文本也能解释目标 patch, 就施加 margin 惩罚, 提高类别语义区分度。
变量说明: \(f_\theta\) 与正目标预测共用同一个 JEPA predictor。\(c^{det}\) 表示 mask 后视觉上下文 detach, 负文本损失不更新视觉 context; \(t_{neg}\) 是负类别文本经过 embed_text 后的特征; \(\hat z_n=f_\theta(c^{det},t_{neg})\) 是用负文本预测出的目标特征; \(s_+=\cos(\hat z,z_m)\) 是正文本预测 \(\hat z\) 与 masked target \(z_m\) 的余弦相似度, 在负文本 loss 中 detach 后作为 margin 参照。

模块说明

CLIP ViT-L/14 @ 336px frozen
视觉编码器和文本编码器均冻结, 保留 CLIP 原始零样本能力。视觉端输出 CLS 与 576 个 patch token, 文本端输出 200 类文本原型。
Text Adapter 可训练
文本侧 bottleneck MLP, adapter_ratio=0.2, 在 seen 训练监督下微调类别文本原型, 同时由拓扑 loss 约束不破坏类别几何。
频域局部补丁选择器 核心模块
使用频域低通差异对 patch 打分, 当前 lastvit_select_k=32, 作为局部信息瓶颈。该模块只负责选择局部 token, 不替换 CLS, 也不进入梯度链路。
FAE 几何感知编码 use_fae=True
对选中的局部 patch 进行几何感知自注意力, 缓解局部 token 位置纠缠, 再送入双向视觉-文本交互。
Cross-Modal Transformer 双向
包含 v2s 与 s2v 两条 cross-attention 分支, tf_common_dim=512, tf_heads=4, 两路分数按 weight_s2v=0.5 融合。
条件文本适配 核心模块
用图像 CLS 生成小幅文本残差, conditional_text_ratio=0.005, 让每张图拥有轻量条件化的 200 类文本原型。
AG-JEPA 训练辅助
用真实类文本选择 top-8 semantic patch 作为 masked target, 以剩余 patch 与类别文本预测目标特征。推理时不走 JEPA predictor。
Add 融合输出 score_mode=add
最终分类分数为 logits = base_logits + 0.3 · local_score。全局分支提供 CLIP 兜底, 局部分支补充细粒度判别。
