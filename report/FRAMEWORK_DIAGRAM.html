模块说明

CLIP ViT-L/14 @ 336px frozen
视觉 + 文本编码器, 完全冻结不训练。视觉端输出 [B, 577, 768] (CLS + 576 patches), 文本端输出 [200, 768] 类原型。
Adapter (text) ~295K 参数
文本侧 bottleneck MLP 768→192→768, residual ratio=0.2。整个网络唯一的主体可训练参数, 用于精炼 seen 类文本特征。
★ G3 CoCoOp Conditional Text 主创新
图像条件文本残差: π(x)=meta_net(v_cls), t̃_c(x)=Norm(t_c + 0.005·π(x))。让每张图都有自己的 200 类文本原型, 同类不同图侧重不同属性。
★ LaSt v5 Patch Selector 主创新 · 0 参数
频域前景筛选: FFT + 高斯低通(σ=√D), 算每个 patch 的"通道平稳性"分数, topK=32 选最平稳的前景 token (滤背景 / artifact / lazy aggregation)。不进梯度链路, 显存零增长。
FAE 子集 gather 几何感知
对选中的 32 个 patch 做几何感知自注意力。从全图 box_emb [576,576,64] 做 advanced indexing 取 [B,32,32,64] 子矩阵, attention 计算量 = 全图的 1/81。
Cross-Modal Transformer 双向
两条 cross-attention 路径并行: v2s (文本 query patch) + s2v (patch query 文本), 公共维 dim=512, heads=4。两路 logits 等权融合得到 local_score。
★ AG-JEPA 训练辅助 · 主创新
用真实类文本与 patch 做相似度选 top-8 作 masked target, predictor 用剩余 patch + 类文本在抽象空间预测 target 特征 (MSE)。配反事实负类文本 margin loss 防"非因果背景"被当成属性表征。仅参与 loss, 不影响推理路径。
最终融合 (add 模式) 输出层
全局 base 分支 + 局部 local 分支线性叠加: logits = base_logits + 0.3 · local_score。base 给 unseen 一个"CLIP 兜底", local 在 seen 上做精细化加分。
G1 Loss-level: Topology + Consistency 软正则
Topo Pearson: 拉近 Adapter 输出与原 CLIP 文本两两 cosine 矩阵的 Pearson 相关性, 防 seen-only CE 拉散 unseen 几何。Cons KL: base ↔ local 分布 KL 拉近 + 动态衰减防爆。
G2 MSDN++ Mutual Distillation 软正则
双向 KL: s2v 教 v2s, v2s 也教 s2v (两边都 detach), 让两条 cross-attention 路径互相对齐, 防一支塌掉。T=2, 权重 0.05。
