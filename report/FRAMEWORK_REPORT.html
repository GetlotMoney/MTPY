DVSR / VGSR 当前框架汇报

基于 CLIP 的细粒度广义零样本学习 (GZSL) · CUB-200-2011 · CLIP ViT-L/14@336px frozen
汇报时间: 2026-06-01 · 版本: P3.10 配置 (60ep + AG-JEPA + LaSt v5 K=32 + PriorCorrection OFF)

一、研究问题与定位

本工作针对 GZSL 在 CLIP 主干上的两个核心痛点:

  1. CLIP "近视": ViT-L/14 在细粒度任务上倾向于全局轮廓, 容易忽略局部判别属性 (鸟喙形状、翅纹细节)。
  2. CLIP "偏心": 训练时只见过 seen 类监督, 测试时严重偏向 seen, unseen 准确率受压制。

在 frozen CLIP ViT-L/14@336px 基础上, 通过一个轻量 Adapter (~295K 可训练参数) 加上视觉前端 patch 选择 + 训练期辅助 JEPA + 文本侧软正则的组合, 在不微调主干的前提下显著缓解上述问题。

当前 main result (3-seed): H = 72.65 ± 0.19 vs 纯 CLIP zero-shot 61.28, 累计 +11.37

二、整体框架数据流

                  ┌─────────────────────────────────────────┐
                  │   CLIP ViT-L/14@336px  (完全 frozen)    │
                  └─────────────────────────────────────────┘
                       │                              │
                       │ image → CLS+576 patches      │ 200 类 GPT-5.5 描述
                       ▼                              ▼
                  [B, 577, 768]                   [200, 768]
                       │                              │
                       │                              ▼
                       │                  ┌─────────────────────────┐
                       │                  │ Adapter (768→192→768)   │ 唯一可训练
                       │                  │ ratio=0.2 残差          │ ~295K 参数
                       │                  └─────────────────────────┘
                       │                              │ t_c
                       │                              │
                       ├── CLS [B,768] ── base 路 ───→ base_logits = γ·cos(v_cls, t_c)
                       │                              ▲
                       │                              │ G3 CoCoOp 条件文本残差
                       │                              │ t̃_c(x) = Norm(t_c + 0.005·π(x))
                       │
                       ▼ 576 patches
              ┌────────────────────────────────────────┐
              │ ★ LaSt v5 频域 patch 选择器 (零参数)    │
              │  FFT → 高斯低通(σ=√D)                  │
              │  S_i = mean( |x| / (|x_lp - x| + ε) )  │
              │  topK(K=32, largest=True)              │  → 选"通道平稳前景"token
              └────────────────────────────────────────┘
                       │ patches [B, 32, 768]
                       ▼
              ┌────────────────────────────────────────┐
              │ FAE 子集 gather (1/81 计算量)          │
              │ box_emb 子集 [B, 32, 32, 64]           │
              │ Geometry-aware Self-Attention          │
              └────────────────────────────────────────┘
                       │ memory [B, 32, 512]
                       ▼
              ┌────────────────────────────────────────┐
              │ CrossModalTransformer (双向)            │
              │   v2s 路 + s2v 路                       │
              │   weight_s2v = 0.5                      │
              └────────────────────────────────────────┘
                       │
                       ▼
              local_score = 0.5·s2v + 0.5·v2s   [B, 200]
                       │
                       ▼
        ┌──────────────────────────────────────────┐
        │  最终融合 (add 模式)                     │
        │  logits = base_logits + 0.3·local_score  │
        └──────────────────────────────────────────┘
                       │
                       ▼
                 [B, 200] 全类 logits → argmax → 预测类

训练期旁路 ★ AG-JEPA (不进 forward 主路径, 仅参与 loss)

真实类文本 t_y  ──→ patch-text 相似度  ──→ top-8 masked target patches
其余 patches + t_y  ──→ jepa_predictor (MLP)  ──→ 预测 masked 区域特征
                                          │
                                          ▼ MSE → loss_jepa (λ=0.05)
负类文本 t_neg ──→ 同 predictor → margin → loss_jepa_neg (λ=0.02, margin=0.2)

三、核心模块清单

# 模块 角色 关键超参 训练参数
1 CLIP backbone 视觉 / 文本编码器 (frozen) ViT-L/14@336px, 768 维, 576 patches 0
2 Adapter (text) seen 类文本特征精炼 768→192→768, residual ratio=0.2 ~295K
3 G3 Conditional Text Adapter CoCoOp 图像条件文本残差 π(x)=meta_net(v_cls), ρ=0.005 ~75K
4 ★ LaSt v5 Patch Selector 频域前景 token 选择 (零参) K=32, σ=√D, formula=v2_abs_mean, largest=True 0
5 FAE 子集 gather 几何感知自注意力 (子集索引) 32×32 子矩阵 advanced indexing 复用
6 CrossModalTransformer 双向 v2s + s2v cross-attention dim=512, heads=4, dropout=0.1 ~
7 ★ AG-JEPA (训练期辅助) masked patch prediction + 反事实负样本 top-8, hidden=512, λ=0.05, λ_neg=0.02, margin=0.2 ~
已验证关闭模块 (踩过坑, 不再启用): D-α 动态门控 / D-w 双分支动态 / DMP 动态池化 / L_bias hinge / Calibration loss / L2-SP / LaSt-CLS 残差 / cosine_only 模式 / PriorCorrection balanced (在 CUB 经控变量验证 hurts -0.21 H)

四、损失函数

L_total = L_CE_seen                                          ← 主分类
        + 0.05 · L_topo_pearson                              ← G1 文本拓扑保持
        + 0.05/(1 + 0.1·cons) · L_consist (KL base↔local)    ← G1 一致性 + 动态衰减防爆
        + 0.05 · L_msdn (KL s2v↔v2s 双向 detach, T=2)        ← G2 双分支互蒸馏
        + 0.05 · L_jepa (top-8 masked patch MSE)             ← AG-JEPA 主项
        + 0.02 · L_jepa_neg (margin=0.2 反事实)              ← AG-JEPA 反事实约束

八、框架特性总结

  1. 零主干微调: CLIP ViT-L/14@336px 全程 frozen, 仅 Adapter (~295K) + 几个轻量 MLP 可训练
  2. 零参数 patch 选择: LaSt v5 用 FFT + 高斯低通做离散 indices, 不进梯度链路, 显存零增长
  3. 训练加速: FAE 在 32 个选中 patch 上 self-attn, 计算量 = 全图 576 的 1/81
  4. 训练期辅助 JEPA: AG-JEPA 仅参与 loss, 不影响推理路径, 推理速度与 baseline 一致
  5. 多 seed 稳健: 3-seed std=0.19, 跟旧 main 0.11 同量级, 无离群点
  6. 控变量纪律: 每个主创新都有干净对照组, 边际贡献可独立验证
