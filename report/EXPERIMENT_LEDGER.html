二、每次实验详细记录

实验 #1 — T1 strict baseline (main result)

框架CLIP frozen + Adapter (768→192→768) + FAE 双分支 + mean pool + add 模式
损失L_CE + 0.05 L_topo + λ_eff L_consist + 0.05 L_msdn (G1+G2+G3 全开)
关键模块 pool_method=mean, gating=fixed, gating_dynamic=fixed, weight_s2v_mode=fixed, pool_dynamic=fixed, use_conditional_text=True (ratio=0.005), score_mode=add, local_weight=0.3
训练random_seed=5, batch=64, Adam lr=1e-3 wd=1e-4, lr_stages=[1e-3×20, 1e-4×30] strict 严格连续
结果Best @ ep34: H=72.27 / U=72.74 / S=71.80 / ZSL=81.02
日志training_log_CUB_2026-05-30_00-56-54.txt
ckptbest_model_CUB_2026-05-30_00-56-54_H7227.pth
结论论文 main result, 后续所有 ablation 都用同一 schedule 保证公平

实验 #2 — T1 warm-restart 对照

框架同实验 #1
差异训练流程改为 warm-restart: 第 1 段 20ep 选 best ckpt → 第 2 段加载 best 后 finetune 30ep
结果Best @ ep22: H=73.05 / U=69.80 / S=76.61 / ZSL=81.57
日志training_log_CUB_2026-05-29_23-00-51.txt (Stage 2)
training_log_CUB_2026-05-29_22-55-09.txt (Stage 1)
ckptbest_model_CUB_2026-05-29_23-00-51_H7305.pth
结论warm-restart vs strict +0.78 H, 不是 noise 是机制差异. 论文 appendix 对照"best-checkpoint selection effect"

实验 #3 — T2 score 级动态门控 (strict)

框架同 T1 + alpha_net (768→64→1, ~50K 参数)
关键改动gating_dynamic: fixed → mlp
forward α(x) = sigmoid(alpha_net(cls_token))
logits = base_logits + α(x) · local_score
训练同 T1 strict, 仅多了 alpha_net
结果Best @ ep36: H=71.65 / U=74.50 / S=69.02 / ZSL=81.81
诊断 alpha_net 末期 bias=-0.683 (sigmoid≈0.336), 跟 init 0.3 几乎不动. mlp 退化成常数, 没学到逐图差异
日志training_log_CUB_2026-05-30_10-40-38.txt
结论 不正向vs T1 strict (-0.62). T2 在 strict schedule 下不显著, 待 warm-restart 验证

实验 #4 — T2 score 级动态门控 (warm-restart) 当前任务

框架同 T2

ep21 自动加载 Stage 1 best ckpt 后切 lr=1e-4 cosine T_max=30
