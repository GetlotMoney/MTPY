四、损失函数公式总表

损失 公式 关键设计 实测增益
L_CE F.cross_entropy(logits[:, seen], seen_labels) 仅 seen 150 类 主损失
L_topo 1 − Pearson( PWC(t_enh), PWC(t_clip) ) 锁相对几何而非绝对距离 +0.37 H
L_consist T²·KL( softmax(base/T) || softmax(local/T) )
λ_eff = 0.05/(1+0.1·L.detach())		 仅 seen 列, base.detach(), 倒数衰减 +0.5 (软对齐双路)
L_msdn (T²/2)[KL(p_s2v.detach()||p_v2s) + KL(p_v2s.detach()||p_s2v)] 双向对称, .detach() 防梯度环 +0.18 H
总 Loss L_CE + 0.05·L_topo + λ_eff·L_consist + 0.05·L_msdn 4 项相加 72.24 → 73.05
