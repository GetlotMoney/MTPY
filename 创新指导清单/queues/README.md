# 分类实验待做清单

这个目录保存各类实验的长期待做清单。它位于创意树和 `backlog.md` 之间。

层级关系：

```text
创意树：长期想法库
  ↓
分类待做清单：每类实验自己的候选队列
  ↓
backlog.md：当前执行窗口，只放近期 3-10 个实验
  ↓
experiments/：已经执行或正在执行的实验记录
```

文件职责：

| 文件 | 职责 |
|---|---|
| `01_module_replacement.md` | 创新模块、模块替换、结构性改动。 |
| `02_ablation.md` | 消融实验，验证已有模块是否必须保留。 |
| `03_hyperparam_tuning.md` | 超参、loss 权重、训练日程、top-K、margin 等扫描。 |
| `04_cross_dataset.md` | AWA2、SUN 等迁移与泛化实验。 |
| `05_final_runs.md` | 最终 seed 候选池、正式复核、热重启上限或附录结果。 |

使用规则：

- 长期候选放在分类队列，不放进 `backlog.md`。
- `backlog.md` 只同步当前最想跑的少量实验。
- 每个队列都使用 `P0/P1/P2/P3` 优先级。
- 每个实验项必须可验证、可度量、可回滚。
