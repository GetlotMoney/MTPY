# 调参参数总表（固定框架版）

本表只比较同一套 CUB GZSL 主框架，不混入不同模块结构的实验。

固定模块框架：

- 数据集：CUB GZSL
- 运行口径：seed=5
- 固定主框架：`lastvit_select_k`、`lambda_topo_pearson`、`lambda_msdn`、`lambda_jepa`、`lambda_jepa_neg`、`conditional_text_ratio`、`local_weight`
- 不纳入本表的结构变更：`use_ag_jepa`、`use_cf_neg_text`、`gating_dynamic`、`adapter_ratio`、`lr_stages` 等

比较规则：

- 同一框架组内再看高低
- 不同模块结构的实验不要直接横比
- `H` 为最后一列实验值
