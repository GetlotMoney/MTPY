# 50实验调参计划 — SGDR 三段式

> 策略：三段各30ep SGDR，eta_min不到0。全部 restart_from_best: False
> 每个实验 ~90ep，预计 ~2h/个。串行50个约100h。
> 旧作弊实验 (TUNE-024~033) 保留存档于原目录，不作废。

---

## 通用配置模板

```yaml
lr_stages:
  value:
    - {lr: 0.001,  epochs: 30, eta_min: 1e-5}
    - {lr: 0.0001, epochs: 30, eta_min: 1e-6}
    - {lr: 0.00001, epochs: 30, eta_min: 1e-7}
random_seed:
  value: 5
```

## 执行优先级

1. **P1** — SGDR基线 (TUNE-034~036)
2. **P2** — 轴向搜索，快速定位最佳子区间
3. **P3** — 2D交互网格（基于P2调整范围）
4. **P4** — 二级参数微调
5. **P5** — eta_min 灵敏度确认
6. **P6** — 边界探测

每批跑完后分析结果，调整下一批的范围。
