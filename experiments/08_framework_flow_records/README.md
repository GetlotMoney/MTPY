# 实验代码框架图记录

这个目录集中保存每个实验完成后的代码框架图、实验流程图、指标数据和日志来源。

每个实验一个文件：

```text
<EXP-ID>_<slug>.md
```

示例：

```text
ABL-001_disable_patch_selection.md
```

## 文件必须包含

1. 实验信息：ID、名称、日期、分支、commit、配置路径。
2. 流程图：使用 Mermaid 画出当前代码框架或该实验涉及的数据流。
3. 流程说明：说明这张图展示的是什么流程。
4. 改动定位：说明本实验改变了流程图中的哪一个节点或边。
5. 数据记录：seed、U、S、H、ZS、最佳轮次、原始日志路径、实验日志副本路径。
6. 结论：说明实验结果如何改变对当前代码框架的理解。
